SortedStack Program Explanation
File Name: SortedStack.java

Description:
This program reads a list of integer numbers from the standard input, 
stores them in a Stack, sorts the stack from smallest to largest, and 
displays the sorted stack.

Usage:
To compile and run the program, follow these steps:

Compile the program:
Open your command line interface and navigate to the directory where the
file is located. Then, use the following command:

-javac SortedStack.java

Run the program:
After successful compilation, run the program using:

-java SortedStack

To generate Javadoc documentation:
You can create documentation for the program using:

-javadoc -d doc SortedStack.java
